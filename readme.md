# Go 实现的营销微信机器人

# 研发初衷
家里人开了一个店铺，我想到一个营销好方法：通过吸引客户邀请朋友进群即可获得奖励。但是苦于没有实现统计群成员邀请人数这个功能的程序，于是决定为家里人出一份力，自己动手。

## 部署说明
clone 项目到本地，然后进入项目目录，执行下面的指令

```
go mod tidy //下载依赖

go run main.go // 运行程序
```

## 功能介绍
初步实现了两个功能：
- 统计每一位群成员邀请新成员入群的个数
- 定时向群中推送营销信息（顺便实现了定时给女朋友早安晚安）
- 可以回复一些简单问题，如店铺介绍等
